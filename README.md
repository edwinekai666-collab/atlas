# Atlas

Atlas 是一个面向设计师的个人视觉参考与能力收藏库。

## 本地预览

直接打开 `index.html` 即可使用。素材、标签和编辑内容默认保存在当前浏览器的 `localStorage` 中。

## 发布到 GitHub Pages

1. 在 GitHub 创建一个空仓库。
2. 把本项目推送到 `main` 分支。
3. 在仓库的 **Settings → Pages** 中将发布来源设为 **GitHub Actions**。
4. 推送完成后，工作流会自动发布网站。

GitHub Pages 只负责托管这个前端页面，不会把你的本地收藏自动同步到云端。跨设备使用前，可以在页面中导出 JSON，再在另一台设备导入。
